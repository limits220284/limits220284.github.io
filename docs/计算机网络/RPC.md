

# Rpc

### 1、结构图

![image-20230810111051116](https://s2.loli.net/2023/12/16/yTcsKSxhfIOq39o.png)

- 客户端以本地调用的方式调用服务
- client stub接收到调用后负责将方法，参数组装成能够进行网络传输的消息体，(序列化操作)
- Client stub找到服务端地址，将消息发送到服务端
- server stub接收到消息后进行解码，根据解码结果调用本地服务
- 本地服务执行完毕后返回给server stub，server stub将结果打包成消息发送给客户端
- client stub接收到消息后进行解码然后得到结果(反序列化操作)

总结来说一共包含三个部分***call ID映射***，***序列化和反序列化***，***网络传输***
