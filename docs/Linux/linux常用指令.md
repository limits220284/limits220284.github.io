# Linux常用指令

## 网络监控

1. **netstat/ss -tunlp**   : t 表示tcp，u表示udp，l表示listen，n表示使用ip和端口展示，p展示pid
2. **iftop**：监控网络带宽的使用情况
3. **ifconfig**: 查看本地ip地址信息

## 系统管理

1. **top** ：监控内存和cpu使用率的命令
   - 过程中按P代表按照CPU使用率降序排序
   - M代表按照Memory降序排序
   - 按下1显示所有cpu的核心使用情况
2. **free**：命令用于显示系统的空闲、已用、交换空间等内存信息

   - `-b`：以字节为单位显示内存使用情况。
   - `-k`：以KB为单位显示内存使用情况（这是默认设置）。
   - `-m`：以MB为单位显示内存使用情况。
   - `-g`：以GB为单位显示内存使用情况。
   - `-h`：以人类可读的格式（自动选择适当的单位，例如KB、MB或GB）显示内存使用情况。
   - `-t`：显示内存总计行（即总计物理内存+交换空间）。
3. `vmstat`（Virtual Memory Statistics）命令是Linux系统中用于显示关于虚拟内存、进程、CPU活动等信息的工具。它提供了系统的即时状态，包括内存使用、系统进程、中断、CPU活动和交换空间使用情况。`vmstat`可以帮助识别系统的性能瓶颈，特别是内存和CPU相关的问题。
   - Procs（进程）
     - `r`：等待运行的进程数。
     - `b`：处于不可中断睡眠状态的进程数。
   - Memory（内存）
     - `swpd`：使用的交换空间大小。
     - `free`：空闲的物理内存大小。
     - `buff`：用作缓冲的内存大小。
     - `cache`：用作缓存的内存大小。
   - Swap（交换）
     - `si`：每秒从交换空间读入内存的数据量。
     - `so`：每秒从内存写入交换空间的数据量。
   - IO（输入/输出）
     - `bi`：每秒从块设备读入的数据量。
     - `bo`：每秒向块设备写入的数据量。
   - System（系统）
     - `in`：每秒的中断次数，包括时钟中断。
     - `cs`：每秒的上下文切换次数。
   - CPU（中央处理器）
     - `us`：用户模式下CPU的时间百分比。
     - `sy`：系统（内核）模式下CPU的时间百分比。
     - `id`：空闲CPU的时间百分比。
     - `wa`：等待IO的CPU时间百分比。
     - `st`：被偷走的时间百分比（对于虚拟机环境）。
   - cat /proc/vmstat | grep 'pswpin\|pswpout' ：可以用来统计内存交换的次数
4. **systemctl**：控制systemd系统和服务管理器。

   - 启动服务：`systemctl start servicename`
     
   - 停止服务：`systemctl stop servicename`
     
   - 重启服务：`systemctl restart servicename`
     
   - 显示服务状态：`systemctl status servicename`
     
   - 开机自启：`systemctl enable servicename`
     
   - 禁止开机自启：`systemctl disable servicename`
5. **ps**：显示当前所有的进程
   - -e显示所有进程
   - -f表示全格式
   - a表示显示除控制进程（session leaders）和没有控制终端的进程外的所有进程
   - u表示用户为中心的格式输出，提供每个进程的用户名称，CPU和内存使用率等详细信息
   - x显示没有控制终端的进程，通常与进行一起使用
6. **kill -9 pid**: -9强制终止进程，不加的话就是终止进程
7. **killall**：根据进程名终止所有进程
8. **jobs**：列出当前会话中的所有任务，包括后台运行的进程。
9. **pkill**：以特定的模式杀死进程
   - -f 通过完成命令行匹配进程，不仅仅是进程名，必须有完整的cmd
   - -u 指定要匹配的用户的用户名或用户ID
   - -x 只匹配与指定模式完全匹配的进程，不匹配存在字符串的进程
10. **nice**：优先级范围从-20（最高优先级）到19（最低优先级)
   - nice -n 5 command：表示nice值 + 5启动command，通常越小，优先级越高
11. **journalctl** : 查看系统所有日志
   - -u 表示查看特定服务的日志
   - --since today 表示查看今天的日志
   - -f 滚动查看最新的日志
   - --since "2021-01-30 14:00" --until "2021-01-30 16:00" 查看某个时间段的日志
11. **lscpu**：查看cpu信息
12. **getconf**：可以获取系统变量的值
    - LONG_BIT：表示长整型的bit数目
13. **df**：显示文件系统的磁盘空间信息
    - -h：人类可读；
    - -a：显示所有文件系统的详细信息，包括特殊文件系统；
    - -l：仅显示本地文件系统，不包括通过网络挂载的文件系统；
    - -i：显示文件系统的inode的使用情况，包括已用inode数量、剩余inode数量以及inode使用率；

## 文件操作

1. **pwd**：显示当前文件目录
2. **mv**： 可以进行文件命名操作，也可以进行文件移动操作
3. **ls**：显示文件或者文件夹的信息
   - -a 显示所有文件和目录，包括以.开头的
   - -l 以详细格式列出文件和目录的信息，权限、链接数、所有者、组、大小、时间戳、名称
   - -d 显示文件夹本身，而不是它们包含的信息
   - -h (human readable)人类更容易阅读的方式
4. **tr** 命令的一般语法如下：
   
   - tr [选项]... SET1 [SET2] 代表将set1中的字符转成set2中的字符
     - -d 表示删除set1中存在的字符，**一般不能够用来删除汉字，因为汉字字节数不是1**
     - -s 压缩set1中连续存在的字符，中文不行，底层应该是utf-8，所以导致汉字的编码不是一个字节
     - -c 表示求补集，通常用来删除，不在set1中的字符
5. **uniq** 用来删除相邻的重复项
   - -c统计字符频率
6. **sort**：对字符进行排序操作，一般是字典序
   
   - -c：反向排序；
   
7. **tail**：反向输出文件
   - -n + x 表示从第几行开始输出；
   - tail -n x 表示查看后面x行的日志；
   
7. **head**：查看文件开头
   
   - -n x 表示查看开头x行；
   - -n -x 表示查看除了最后x行的日志；
   
9. **find**：find \[路径\]\[表达式\]是一个强大的用于在Linux和Unix系统中查找文件和目录的命令行工具
   
   - -name：按照文件名匹配查找
   - -type：根据文件类型匹配查找，如文件('f')、目录('d')等；
   - -size：根据文件大小匹配查找
   - -mtime、-atime、-ctime：根据文件修改时间、访问时间、状态改变时间匹配查找；
   - -exec：对查找到的文件执行指定的命令；
   
   ```shell
   find /home -type f -mtime -30 -size +1M -exec ls -l {} \;
   ```
   
10. **grep**：文件搜索命令

   - `-i`：忽略大小写。
   - `-v`：显示不匹配的行。
   - `-n`：显示匹配行的行号。
   - `-r` 或 `--recursive`：递归搜索子目录。
   - `-A num`：显示匹配行后的 num 行上下文。
   - `-B num`：显示匹配行前的 num 行上下文。
   - `-C num`：显示匹配行前后的 num 行上下文。
   - **-o**：仅输出匹配到的部分，**-E**：启用扩展的正则表达式

11. **sed**：这个命令可以查找日志文件特定的一段 , 根据时间的一个范围查询，可以按照行号和时间范围查询

    - 按照行号

    ```javascript
    sed -n '5,10p' filename 这样你就可以只查看文件的第5行到第10行。
    ```

    - 按照时间段

    ```javascript
     sed -n '/2014-12-17 16:17:20/,/2014-12-17 16:17:36/p'  test.log
    ```

12. **less**：查找文件内容

    - shift + G 命令到文件尾部
    - ？可以查询你想要查询的关键字
    - n向上查找关键字
    - shift + n反向查找关键字
    
13. **echo**：用于在终端命令行中输出文本内容或者变量的值

    - echo [options] [string]
    - -n: 禁止在输出末尾添加换行符；
    - -e: 启用解释转移字符，可以将转义序列解释为特殊字符(如\n表示换行符)；
    - -E: 禁止解释转义字符，将转义字符原样输出；
