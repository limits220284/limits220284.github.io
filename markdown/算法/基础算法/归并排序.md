# 归并排序

- 确定分解点
- 递归
- 将两个有序的数组合并成一个有序的数组

``` python
n = int(input())
nums = list(map(int, input().split()))
tmp = [0] * n

def merge_sort(l, r):
    if l >= r:
        return
    mid=(l + r) // 2
    merge_sort(l, mid)
    merge_sort(mid+1, r)
    k, i, j = 0, l, mid+1
    while i <= mid and j <= r:
        if nums[i] <= nums[j]:
            tmp[k] = nums[i]
            k += 1
            i += 1
        else:
            tmp[k] = nums[j]
            k += 1
            j += 1
    while i <= mid:
        tmp[k] = nums[i]
        k += 1
        i += 1
    while j <= r:
        tmp[k] = nums[j]
        k += 1
        j += 1
    nums[l:r+1] = tmp[0:k][::]

merge_sort(0, n-1)
for i in range(n):
    print(nums[i], end=' ') 
```
