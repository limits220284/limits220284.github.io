## [1、区间和](https://www.acwing.com/problem/content/804/)

# 离散化

- 整数
- 有序
- 将过大的操作点映射到连续的区间中，然后再进行操作

```python
m, n = map(int, input().split())
alls = []
adds = []
query = []
#首先进行离散化操作
def find(x):
    l, r = 0, tot-1
    while l < r:
        mid = (l + r) >> 1
        if alls[mid] >= x:
            r = mid
        else:
            l = mid + 1
    return l + 1
    

for i in range(m):
    x, y = map(int, input().split())
    adds.append([x, y])
    alls.append(x)
for i in range(n):
    l, r = map(int, input().split())
    query.append([l, r])
    alls += [l, r]
#对alls进行排序加去重
alls = list(set(alls))
alls.sort()

tot = len(alls)
a = [0] * (tot+1)
s = [0]
#进行加操作
for t in adds:
    x = find(t[0])
    a[x] += t[1]

#求前缀和
for i in range(1, tot+1):
    s.append(s[i-1] + a[i])

#进行查询操作
for t in query:
    l, r = find(t[0]), find(t[1])
    print(s[r]-s[l-1])
    
```
