## [907. 子数组的最小值之和](https://leetcode.cn/problems/sum-of-subarray-minimums/)

# 单调栈

- 适用于找到某个数左边或者右边最近的`最大的数` 或者`最小的数`

```python
n=int(input())
nums=list(map(int,input().split()))
stk=[-1]
for i in range(n):
    while stk and stk[-1]>=nums[i]:
        stk.pop()
    print(stk[-1],end=' ')
    stk.append(nums[i])
```
