# 基础知识

## 概念

```shell
echo $shell # 系统环境变量
echo $0 # 输出当前执行脚本的名称
```

## Tips

1. `$((...))` 是用于进行算术运算的语法结构，称为算术展开（Arithmetic Expansion）。它用于计算括号中的表达式，并将结果返回。
2. 赋值的时候，等号两边不能加空格；
3. export 定义的环境变量只在当前会话下有效；
4. 单引号中的内容会被视为纯文本，不会对其中的变量、命令替换和特殊字符进行解释或展开。
5. 双引号中的变量会被解析和展开为其对应的值。例如，`"$var"` 将会把变量 `var` 的值插入到字符串中。
