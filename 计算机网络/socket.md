# socket编程

- 常用的socket类型有两种: 流式socket和数据报式socket，流式是一种面向连接的socket，针对于面向连接的TCP服务应用，数据报式socket是一种无连接的socket，针对于无连接的UDP服务应用；





## socket服务器和客户端工作流程?



### 服务器端的工作流程

1. 创建socket：服务器端会先创建一个socket，用于监听来自客户端的连接请求；
2. bind：服务器端将socket绑定到一个指定的网络地址和端口，从而监听来自于这个地址和端口的请求；
3. Listen：监听绑定地址上的请求，等待客户端的连接请求；
4. accept：接受连接，当一个客户端请求连接的时候，服务器端的socket接受请求并建立一个新的socket连接，专门用于与该客户端进行通信；
5. 数据交换：一旦建立连接，服务器端就可以通过新建立的socket与客户端进行数据读写操作；
6. 关闭连接：数据交换完成后，服务器端关闭与客户端的连接。服务器可以继续监听其他客户端的连接请求；



### 客户端的工作流程

1. 创建Socket：客户端创建一个socket，用于与服务器端的socket进行连接；
2. connect：客户端的socket尝试连接服务器端的socket，需要指定服务器的网络地址和端口号；
3. 数据交换；
4. 关闭连接；
